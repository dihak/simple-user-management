## Simple User Management

Aplikasi Login dan Register sederhana yang dibuat dengan Framework CodeIgniter.

## Features

Fitur :

- Register
- Log In
- Edit User
- Delete User
- ServerSide Datatables
- Create Update Delete Menggunakan Ajax
- Image Upload
